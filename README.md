# MediaTool
基于JavaFX的视频处理图形化工具

``` lua
MediaTool
├── ffmpeg -- 处理视频核心工具
├── lib -- 依赖jar包
├── resources -- 静态资源文件
├── src -- 源文件
```

####准备
- **ffmpeg** 
    - 1.由于ffmpeg文件比较大,所以项目并没有上传,还请各位通过[官网](http://ffmpeg.org/download.html)下载
    - 2.文件放在项目根目录与resources同级目录下
